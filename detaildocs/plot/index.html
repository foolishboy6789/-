# 详细剧情

# 第1集 异能兽入侵

  •  中国东北海面惊现怪兽袭击渔民事件，几乎所有的受害人都失去了某种特定的反应，简单的说就是他们的某段记忆被抹去了。一时舆论哗然，附近的渔民惶惶不安。在这种不安的环境中D市的全国钢琴比赛依旧如期举行，少年炘南一举夺得比赛冠军，顺利成为明日新星。全家准备热热闹闹为炘南庆祝，炘南却秘密来到ERP里的秘密训练基地进行格斗训练，用于对付埋藏在黑暗中的影界基地。

# 第2集 炎龙现身

  •  历史系书呆子吴启东无意之中发现异能怪兽，并被怪兽用粘液袭击。恰巧路过的炘南利用炎龙召唤器发射火晶石之光能量救下吴启东。然而铠甲勇士现身击败怪兽的过程，却被杂志社的记者小嵩抓拍到。炎龙侠一时之间名声大噪，火之铠甲的出现让他的死对头影界基地的霸王怒火中烧，影界布下两计，一是抹去铠甲勇士后人的记忆，二是消灭铠甲。

# 第3集 逃跑的蚱蜢兽

  •  吴启东莫名被新型异能怪兽纠缠上，炘南变身炎龙铠甲勇士挑战怪兽，救下差点丧命的吴启东。但是炘南一人难敌二怪，启东受伤住院。吴启东的住院报告上显示出他和东杉的血缘关系，东杉和美真推断启东有可能也是铠甲勇士的后人。另一方面记者小嵩追查怪兽来源时，意外发现一条重要线索，嘉新公司使用管制污染物硝酸汞……

# 第4集 神秘失踪的马匹

  •  全省钢琴大赛开赛在即，炘南和张健成为最大的竞争对手，然而张健根本没有将炘南放在眼里。炘南比赛的压力巨大，黑暗影界的攻击却越来越频繁。影界另出恶计欲将木之铠甲的传人启东引诱到锅炉厂，然后化了木之铠甲。 还好东杉利用追踪器及时扑捉到启东的位置，东杉赶到锅炉厂时影界的走狗丑将已经将启东五花大绑起来。

# 第5集 奇怪的神山老人

  •  D市的一个工地里，工人正在进行“榔头王”的比赛，突然间从地下窜出一个岩石怪兽，许多工友身负重伤。发现异能的美真及时让风鹰侠东杉出动。较量之中东杉对怪兽的封印失败，怪兽趁机逃脱。东杉带回受伤的工友阿土，美真检测发现受伤的人竟是光影铠甲光影村的后人。可惜阿土血液不纯正，不具备召唤光影铠甲的能力。

# 第6集 东杉：我是风鹰侠

  •  郊区煤厂大量的丢煤，小嵩依照线索追查却遭到工头的阻止。小嵩怀疑黑暗组织在采矿场进行秘密破坏活动。小嵩将自己的疑虑透漏给美真，美真一行迅速展开了调查。东杉和炘南来到矿区调查，发现丑将和他携带的怪兽。双方展开了激烈的决斗，炘南和东杉联手击败了岩石怪兽，却被另一同行而来的怪兽躲过必杀帖逃回影界基地。

# 第7集 新生的魔牛兽

  • 炘南的表妹小兰的生日就要到了，在小兰家饺子馆打工的敏慈知道小兰一心想要一条刊登在杂志上的花裙子，细心的她连夜做了一套一模一样裙子送给小兰，小兰收到后非常高兴。就在大家准备给小兰过生日之际，光影村后人阿土遭到了异能魔牛怪兽的攻击。东杉及时赶到救下阿土，炘南接到消息也迅速赶来协助。二人齐心合力对战异能魔牛怪兽。但却在战斗过程中遭受到重度污染。

# 第8集 封印蚂蚁兽

  •  无辜市民受到蚂蚁怪兽的攻击，东杉及时出手救下受害市民，眼见就要打败怪兽，却被它释放出的黑色雾气笼罩，错失进攻时机被蚂蚁兽溜走。因为异能怪兽总是能够迅速纠缠上光影村的后人，东杉和美真怀疑光影村后人身上拥有可以吸引异能兽的特殊物质，美真决定利用东杉的特殊体质作为实验体。让东杉觉得自己就好像是实验室里任人宰割的小白鼠。

# 第9集 光影后人的秘密

  • 炘南大败蚂蚁怪兽，救下被影界基地锁定的小女孩小细。美真认定影界基地能够快速找到光影村后人，一定是他们身上具备某种特殊的体质。美真请炘南配合自己的检查。却失望的发现炘南和东杉的体质并无相似之处。一个偶然的机会，炘南手中的可乐启发了美真，美真及时检测，发现原来光影村后人体内的气压值大大低于常人。丑将追踪到光影村后人小细，幸亏炘南及时赶到，阻止丑将将小细夺走的阴谋，但他身中怪兽包围。坤中赶到事发现场抱走小细，并通知东杉解救炘南。

# 第10集 出发，黑犀侠上

  • 炘南、东杉大败杏木怪兽和百合兽。丑将见大势已去，仓惶逃跑。逃跑之时还贼心不死的在小细身上放上追踪器。为了保护好小细，炘南将小细带到小姨的幸福饺子馆。影界逼迫丑将再次出动，一定要将小细带回基地研究。丑将趁夜来到饺子馆抓人，炘南将丑将引到室外，却身中丑将喽啰的包围。眼看丑将恶计成功，一辆跑车疾驰而来，车上跃下一神秘男子，击倒丑将一众。来人正是小细的哥哥，黑犀铠甲的召唤者北淼。

# 第11集 出发，黑犀侠下

  • 北淼及时赶到，击溃了异能怪兽脓液兽。从丑将手中救下炘南和妹妹小细。北淼能力很高但为人傲慢，善良的敏慈很看不惯北淼目中无人的样子，两人关系有点尴尬。丑将出师不利，为了将功补过，向影界请罪，主动请求调查北淼的背景资料。北淼觉察到敏慈对自己的不满，特地邀请敏慈吃全市最好最贵的蛋糕赔罪。却被出现的异能兽搅了兴致。

# 第12集 大战恶狼兽上

  •  市区出现异能兽，东杉接到消息后第一时间赶到现场。东杉与此次出现的恶狼兽战斗激烈。恶狼兽行动凶狠灵敏，东杉竟然能量不足，被恶狼兽纠缠打斗的奄奄一息。东杉命悬一线，幸亏闻讯而来的炘南及时相救。正在附近陪同敏慈的北淼也迅速进入战斗状态。影界见三个光影战士同时现身，急忙召回恶狼。美真见北淼能力出众，邀请他一同加入对抗黑暗势力的战斗，但北淼执意要做光影战士团体的领导人。

# 第13集 大战恶狼兽下

  • 美真为北淼准备好适合的召唤器以及变身石，北淼正式加入铠甲勇士。市民魏爷爷只是一个普普通通的老头，在街边开了一个小卖部维持生计。坤中无意中发现魏爷爷有一块纹路奇特的怪石。魏爷爷说那是自己年轻时干苦力，无意中发现的。在与坤中交谈后的当天晚上，怪石不断发光，惊惶失措下的魏爷爷报警。闻讯而来的记者小嵩对魏爷爷进行了专访。恶狼兽不期出现，北淼、炘南、东杉三人合力围攻，并使出“禅定印”将其封杀。

# 第14集 土影石出世

  •  魏爷爷拥有怪石的事情，被小嵩刊登在杂志的头版头条上。这块怪石一出现便引起了铠甲勇士和影界基地正邪两方的注意。两股力量怀疑这块怪异的石头，就是传说中的光影石。美真和东杉特意来到魏爷爷的小店，对石头偷偷做了鉴定。确认怪石确实是土影石。影界秘密确认之后也让丑将行动，务必抢到土影石。丑将设计偷走影石，被北淼和东杉截住。影石争夺激战之时，西钊和冰儿意外出现，阻挠北淼和东杉的行动，影石不幸落入影界手中。

# 第15集 被夺走的土影石

  •  得到土影石的影界，妄图独吞土影石。但是他的阴谋很快被界王发觉，界王下令除掉影界以绝后患。丑将见影界大势已去，主动请缨执行杀死影界的任务。丑将逼迫影界交出土影石，影界仰仗影石的威力，根本不把丑将放在眼里。西钊现身召唤雪獒铠甲，对抗影界的地虎铠甲。影界被西钊打回原形，仓惶逃走。丑将一路追踪，并启用魔蚕兽将影界打的奄奄一息。东杉赶到事发地，赶走丑将救下影界。影界临死之前将土影石托付给东杉。

# 第16集 寻找地虎侠

  •  魏爷爷将东杉误会成偷影石的小偷，对东杉不依不饶。经过东杉的再三解释，魏爷爷了解到事情的来龙去脉，并信任东杉能够好好的对待影石。小嵩无意中发现卖烧烤的小李被蜥蜴怪兽袭击，还好炘南搭救。小嵩以为一旁的西钊只是与事无关的路人，阻止他进入战斗，却被西钊打晕。西钊变身对抗炘南，被炘南打的毫无还手之力。烧烤小李被带回检查，结果发现他并非影村后人，实验室里再次为地虎铠甲以及雪獒铠甲的召唤人发生争执。

# 第17集 地虎侠现身

  • 坤中来到美真的实验室进行仔细的检查，美真惊喜的发现坤中竟然是地虎铠甲的召唤人。被西钊打昏的小嵩住进了医院，在美真的帮助下，小嵩怀疑西钊就是雪獒铠甲的召唤人。小嵩出院后，主编很快给他安排了新的任务，采访八天刊最重要的广告客户，小鸭集团的林总。林总因为曾揭露集团内部的贪污案，在公司内部深得民心。但却引起董事会的不满，董事会的人借故公报私仇，将林总在他生日那天开除。被炒鱿鱼，林总心中非常不快，散心海边，却遭到蛤蟆兽的攻击。

# 第18集 水母兽的突袭

  • 坤中及时赶到，救下受到蛤蟆兽攻击的林总。在这次战斗中，坤中展示出地虎勇士强大的威力。六省的钢琴冠军杨军来到幸福饺子馆，他听了炘南的弹奏后，觉得炘南的琴声中有一种对抗的声音。炘南对杨军的评价若有所思。因为炘南迟迟未参加实验室的特训，北淼大发雷霆，斥责炘南心中只知道练琴。炘南承认钢琴对自己的重要，主动交出团队指挥权。一天夜晚，林总在自己的小区遭到水母兽的攻击，美真及时通知北淼将水母兽封印。

# 第19集 界王的追杀令

  • 坤中加入铠甲勇士，并进入藏在实验室中的虚拟战斗室。小嵩在幸福饺子馆巧遇林总，想对他进行专访。林总接受小嵩的采访，但条件是小嵩必须真实、完整的记录事情的真相。小嵩如实的进行了撰稿，但杂志社的主编不想得罪于人，将小嵩的采访告密给小鸭集团的董事。影界基地里丑将向界王报告西钊和坤中走的很近，并且查明坤中是地虎召唤者，界王下令想办法除掉坤中。

# 第20集 界王的秘密

  •  影界基地放出天牛兽进行破换活动，东杉和坤中合力将天牛兽成功封印。西钊在这场战斗中元气大伤，但却侥幸逃脱。东杉叹息没能成功抓获雪獒，坤中却不忍心下手，他觉得雪獒始终是铠甲勇士的一员。得到主编告密的小鸭集团董事，委托冰儿教训一下林总和小嵩。界王看到关于科学奖的新闻，回想起十年前自己也曾是一名杰出的科学家。因为实验事故失去左眼和右手，无法工作后自暴自弃。黑帝发现他对世间的痛恨，绑架他回到影界基地，并且将他变成界王。

# 第21集 铠甲勇士，战斗吧

  • 北淼劝说坤中能够支持他做铠甲勇士的领导者，坤中勉为其难的答应下来。自从放弃团队的指挥权，炘南练琴时逐渐变得心不在焉。炘南的焦虑让敏慈和蕙姨十分担心。小嵩走在街上，突然遭到冰儿一行人的攻击，小嵩被和冰儿同行的两位少女按住，并且被冰儿抢走了随身包里的录音机。杨阳兴冲冲的去看海子的演唱会，不想门票是假的。只能在演唱会门口叹气，恰巧经过的林总突然发现有只异能兽正向他们的方向袭来。

# 第22集 恐怖的影界基地

  • 北淼和东杉陷入与两只异能兽的战斗中，一直处于上风的他们，由于雪獒的突然加入，难以轻松迎敌。还好纠缠不久之后，坤中加入战斗，将雪獒引开。东杉和北淼成功封印了两只怪兽魔羊兽和魔鹿兽，救出受到攻击的林总。并向他解释为什么频频遭受攻击的原因，因为林总也是光影村的后人。可恶的是林总不久又陷入冰儿的手中，并且承诺以后绝不泄漏小鸭集团的丑闻。但又令人欣喜的是东杉和美真通过统计怪兽频发的区域，发现了搜索影界基地的方法。

# 第23集 界王的圈套

  • 小嵩指点极度追星的杨阳到海子下榻的酒店索要签名，杨阳出师不利。灰头土脸的他遭到了苍蝇兽的攻击，坤中迅速出击，将苍蝇兽成功封印。界王利用美真急于找出自己的心理，故意放出三只异能兽集中在码头一带，美真误以为码头就是界王的老巢。界王却在码头埋伏，准备一举消灭铠甲勇士。美真却并未轻举妄动，但冲动的北淼不顾美真的劝阻，带着东杉来到码头，正中了界王的圈套。

# 第24集 被篡改的卫星密码

# 第29集 第五个铠甲勇士

# 第33集 五行归位

# 第34集 摧毁！界王的老巢

# 第35集 暗影四大天王

# 第38集 黑暗五护法

# 第40集 坤中的指示

# 第41集 调查失火案

# 第45集 铠甲二队登场

# 第47集 调查！帝皇铠甲的召唤

# 第48集 终结黑暗五护法

  •  新金被恶金护法打成重伤，失去了召唤雪獒铠甲的能力。但新的铠甲勇士依旧坚持利用向阳作为诱饵，围攻黑暗护法。新木将再次出现在市区的异能兽，引向向阳所在的仓库。黑暗护法发现向阳的位置，一起来到仓库，却落入新铠甲勇士们为他们设计的圈套。恶火法不惜与向阳同归于尽，幸亏西钊及时救下向阳。恶木法和恶水法狗急跳墙抓住文东，威胁铠甲勇士。与此同时，炘南在敏慈的坦白下，知道父母死亡的真相，下决心与黑暗军团对战到底。

# 第49集 黑暗影帝登场

  •  新火、新土、新水竟然围攻西钊，威胁西钊不要多管闲事。闻讯赶到现场的美真对新的铠甲成员失望至极，当场解除了他们对铠甲的召唤资格。黑帝趁机绑架了向阳，炘南五人重新组合，恢复对铠甲的召唤权。炘南认为黑帝能够绑架走向阳，说明黑暗势力并未消除。原来黑帝是一个乞丐，因为捡到光影帖与黑暗护法签订了出卖灵魂的协议。为了消灭铠甲勇士，黑帝召唤远古四大恶兽，并且利用向阳企图召唤终极铠甲。

# 第50集 苦斗！四大远古异能兽与帝皇侠

  •  黑帝利用远古四大恶兽企图消灭铠甲勇士，铠甲的必杀技对四大恶兽根本不起作用，在战斗中炘南等人的战斗力逐渐下降。敏慈在炘南和向阳的战斗中差点受伤，炘南根本不是向阳的对手，眼看炎龙铠甲就要被销毁。敏慈不顾自身安危，紧紧抱住向阳，向阳渐渐觉察到敏慈的存在，被黑帝控制的意识终于清醒。黑帝见自己大势已去，招走四大恶兽。

# 第51集 最终的战役

  •  黑帝打开了五恶灵当初送的锦囊，谁料锦囊里面就是当年他发下的重誓。五大恶灵再次出现在黑帝的面前并进入了黑帝的身体，黑帝变成终极魔兽黑魔兽。小嵩告诉铠甲勇士，想要取得正义对抗邪恶的最后胜利，炘南、东杉、坤中、北淼和西钊五个人必须团结一致。黑魔兽出现在市中心展开大肆的破坏活动，铠甲勇士们为了阻止黑魔兽的袭击，将它引向外太空，然而黑魔兽竟渐变为巨型兽，一场艰巨的战斗等待着铠甲勇士。

# 第52集 结之真正含义

  •  黑魔兽变换的巨型怪兽十分厉害，铠甲勇士的战斗能量渐渐不足。战斗中西钊和坤中被巨型兽抓住，并被摔向地球。就在两人命悬一线之时，北淼和东杉不顾自身的安危紧紧拽住西钊和坤中。最后炘南也加入到救援行动中，五位铠甲勇士由于坠落的巨大摩擦，变成了一团大火球。黑魔兽抢先回到地球表面，得意洋洋的等待铠甲勇士的毁灭，不料从天而降了一个帝皇侠，原来五位勇士齐心合力终于召唤出了终极铠甲……
